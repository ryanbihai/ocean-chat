# OceanBus 主页设计方案 v2

## 定位

**面向：AI Agent 的构建者和使用者（开发者 + AI 爱好者），不是面向游戏玩家。**

页面风格：干净、现代、偏开发者工具审美（参考 Stripe / Vercel / Linear），
龙虾船长板块独立使用复古航海风，形成视觉反差和记忆点。

## 视觉规范

### 全局
- 背景：深色 (#020b16 或接近)，干净，不花哨
- 字体：系统字体栈，中文 Noto Sans SC
- 间距：宽松，1600px 最大宽度
- 色彩：蓝色系为主 (#38bdf8 / #0ea5e9)，点缀金色仅用于龙虾船长板块
- 卡片：轻微玻璃质感 (rgba + blur)，圆角 12-16px

### 龙虾船长板块（复古航海风）
- 底色：暖羊皮纸渐变 (#faf0dc → #f5e6c8)
- 边框：深棕色 (#8b6914 或类似)
- 点缀：罗盘玫瑰（CSS 或 SVG）、海怪（🦞🐙）、虚线航线
- 字体：标题可考虑衬线，正文保持无衬线
- 整体：在深色页面中像一个被照亮的古老海图

### 协议层卡片
- 简洁现代，与全局风格一致
- 图标使用 emoji 或简单 CSS 形状
- 不要过度设计——开发者信任的是简洁

### 灯塔项目卡片
- 干净卡片，与全局一致
- 每个项目：图标/emoji + 名称 + 一句话描述 + 链接
